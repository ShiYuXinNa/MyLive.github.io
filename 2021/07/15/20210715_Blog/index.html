基于GitHub+Hexo的博客（1）——注册GitHub账户

发表于 2021-07-15 分类于 Blog Valine：

注册GitHub账户

点击链接前往GutHub官网

点击右上角 sign up 注册

[图片GitHub_signup]

填写注册的邮箱，然后点击 Continue

[图片GitHub_mail]

填写账号的密码，然后点击 Continue

[图片GitHub_USERNAME]

是否通过电子邮件接收产品更新和公告，是填y，否填n，然后点击 Continue

[图片GitHub_YorN]

验证，选出旋涡星系

[图片GitHub_Xwxx]

点击 Create account，创建账户

[图片GitHub_cjzh]

点击 Create account，创建账户

[图片GitHub_cjzh]

登录注册的邮箱查看并填写验证码

登录邮箱查看验证码

[图片GitHub_yzm]

输入获取的验证码

[图片GitHub_yzmsr]

点击Skip this for now跳过

[图片GitHub_skip]

点击Skip this for now跳过

[图片GitHub_skip]

接下来可以选择创建第一个项目，点击Start a project或Create repository新建一个仓库

[图片GitHub_Crep]

填写仓库属性，完成创建

[图片GitHub_biuldR]


点击创建后，就产生一个新的仓库，至此完成GitHub的注册。

[图片GitHub_biuldR]
