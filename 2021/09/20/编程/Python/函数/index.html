Python函数

enumerate()

enumerate(sequence, [start=0])
  • sequence: 一个序列、迭代器或其他支持迭代对象。
  • start: 下标起始位置。

Case:

Code :

seq = ['one', 'two', 'three']
for i, element in enumerate(seq):
    print i, element

Out :

0 one
1 two
2 three

 常与for循环搭配，枚举列表的索引和相应元素。

softed()

sorted(iterable, cmp=None, key=None, reverse=False)
  • iterable: 一个序列、迭代器或其他支持迭代对象。
  • cmp: 比较的 一个函数 ，这个具有两个参数，参数的值都是从可迭代对象中取出，此函数必须遵守的规则为，大于则返回1，小于则返回-1，等于则返回0。PS: Python 3.x 中取消了cmp()函数以及该参数！
  • key: 主要是用来进行比较的元素，只有一个参数，具体的函数的参数就是取自于可迭代对象中，指定可迭代对象中的一个元素来进行排序。
  • reverse: 排序规则，reverse = True 降序 ， reverse = False 升序（默认）。

 soft() 与之用法类似，区别是soft()会改变原始输入，而softed() 不会

Case 1:

Code :

a = [5,7,6,3,4,1,2]
b = sorted(a)
print(a)
print(b)

Out :

[5, 7, 6, 3, 4, 1, 2]
[1, 2, 3, 4, 5, 6, 7]

Case 2:

Code :

L=[('b',2),('a',1),('c',3),('d',4)]

# 利用关键字cmp ，对元组元素的第1项进行排序
print(sorted(L, cmp=lambda x,y:cmp(x[1],y[1])))
# 利用关键字key ，对元组元素的第1项进行排序
print(sorted(L, key=lambda x:x[1]))

Out :

[('a', 1), ('b', 2), ('c', 3), ('d', 4)]
[('a', 1), ('b', 2), ('c', 3), ('d', 4)]
