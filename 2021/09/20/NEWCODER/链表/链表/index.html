反转链表(NC78)

我的题解

ListNode* ReverseList(ListNode* pHead) {
    if(pHead==NULL || pHead->next==NULL) return pHead;
    ListNode * pre = pHead;
    ListNode * p   = pHead->next;
    pHead->next = NULL;
    if(p->next == NULL) 
    {
        p->next = pre;
        return p;
    }
    ListNode * pp  = p->next;
    while(pp->next!=NULL)
    {
        p->next = pre;
        pre = p;
        p = pp;
        pp = pp->next;
    }
    p->next = pre;
    pp->next = p;
    return pp;
}

别人的题解

ListNode* ReverseList(ListNode* pHead) {
    //如果链表为空或者链表中只有一个元素
    if(pHead==NULL||pHead->next==NULL) return pHead;
     
    //先反转后面的链表，走到链表的末端结点
    ListNode* pReverseNode=ReverseList(pHead->next);
     
    //再将当前节点设置为后面节点的后续节点
    pHead->next->next=pHead;
    pHead->next=NULL;
     
    return pReverseNode;
     
}

  巧妙的利用递归算法，一步一步回溯。淦，又是废物的一天。

判断链表中是否有环(NC4)

我的题解

bool hasCycle(ListNode *head) { 
    if(head == NULL) return false;
    int n = 10000;
    while(n-- && head->next!=NULL )
        head = head->next;
    if(head->next!=NULL )
        return true;
    else return false;
}

  利用闭环无限循环的特点，在规定时间内一直遍历链表，退出链表时若下一个链表指针不为空，则为闭环。反之则为开环。缺点是对于结点比较少的闭环，依然会遍历规定时间。

别人的题解

bool hasCycle(struct ListNode* head ) {
    struct ListNode *slow = head, *fast = head;
    while ( fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) {
        return true;
        }
    }
return false;
}

  利用快慢指针行走的速度差，当两个指针再次相遇的时候就说明是个闭环，快指针还充当开环的判断。妙啊。

判断链表环入口(NC3)

我的题解

ListNode* EntryNodeOfLoop(ListNode* pHead) {
if(!pHead || !pHead->next) return nullptr;
vector <ListNode *> a;
while(pHead->next)
{
    for(int i=0; i<a.size();i++)
        if(a[i]==pHead) return pHead;
    a.push_back(pHead);
}
return nullptr;
}

  边走边判断，当第二次经过时输出当前结点。

别人的题解

ListNode* EntryNodeOfLoop(ListNode* pHead) {
     
    ListNode* fast = pHead;
    ListNode* slow = pHead;
    while(fast){
        if(!slow->next || !fast->next->next)
            break;
        fast = fast->next->next;
        slow = slow->next;
        if(fast == slow)
        {
            fast = pHead;
            while(slow != fast)
            {
                slow = slow->next;
                fast = fast->next;
            }
            return slow;
        }
    }
    return NULL;
}

  再一次巧妙的利用快慢指针。
