Git的一些常用指令

关于GitHub的指令

克隆仓库

git clone <个人的仓库地址>

查看仓库状态

git status

文件操作

添加文件

git add <文件名/文件路径>

删除文件

git rm <文件名/文件路径>

提交暂存区

git commit -m [提交信息]

查看提交日志

git log

上传远程仓库

git push <远程主机名> <本地分支名>:<远程分支名>
