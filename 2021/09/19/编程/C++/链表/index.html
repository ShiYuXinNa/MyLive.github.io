C++链表

什么是链表

链表是一种常见的数据结构，与火车类似，链表由一个个结点（车厢）链接而成。与数组相比，链表是一种动态的结构。数组在定义时要规定数组大小，而链表不用，可以根据需要开辟新的存储单元。

链表的结构

链表由若干个结点构成，而每个结点又是一个数据结构。每个结点都包含两块，数据域和指针域。

  • 数据域：用于存放用户所需的数据。
  • 指针域：用于存放next指针，并指向下一个结点的地址。

Code：

struct listNode{
    int VData;                    // 存放数据
    struct listNode * Next;      // 存放nect指针
};

头结点与头指针

头结点不是必须的，头结点一般设在第一个元素之前，头结点一般只有指针域存放头指针，一般不定义数据域。头指针一般用head表示，具有标识作用，指向链表的第一个结点的存储空间。如果有头结点，头指针则指向头结点。如果没有头结点，则指向第一个元素，即首元结点。

Code：

// 创建头指针
struct listNode * newList = new listNode;
// 头指针初始化
newList->Data = 0;
newList->Next = NULL;  // 指向空

链表操作

增加结点

增加单个结点

// 创建新的结点
struct listNode * newNode = new listNode;
// 给新结点赋值
newNode->Next = NULL; //设新结点为最后一个结点
newNode->Data = 1;    //数据设置
// 前结点Nect指针指向新结点
newList->Next = newNode;

增加多个结点

// 设定新增结点数
int num = 5;
while(num--)
{
    // 创建新的结点
    struct listNode * newNode = new listNode;
    // 给新结点赋值
    newNode->Next = NULL; //设新结点为最后一个结点
    cin >> newNode->Data; //输入数据
    // 前结点Nect指针指向新结点
    p->Next = newNode;
    // 辅助指针指向新结点
    p = newNode;
}

插入结点

在头指针后插入结点

// 创建pp指针，为原来链表中头指针指向的结点
struct listNode * pp = newList->Next;
// 创建一个要插入的结点
struct listNode * waitNode = new listNode;
waitNode->Data = 99999;
// 头指针指向插入的结点
newList->Next = waitNode;
//插入的结点指向pp结点
waitNode->Next = pp;

在链表中间插入结点

// 假设插入在p结点后，则pp为原来链表中p的后一个结点
struct listNode * pp = p->Next;
// 创建一个要插入的结点
struct listNode * waitNode = new listNode;
waitNode->Data = 99999;
// p结点指向插入的结点
p->Next = waitNode;
//插入的结点指向pp结点
waitNode->Next = pp;

在链表尾部插入结点

// 创建p指针，遍历链表使p指针指向链表尾部
while(p->Next!=NULL)
p = p->Next;
// 创建一个要插入的结点
struct listNode * waitNode = new listNode;
waitNode->Data = 99999;
// 尾指针指向插入的结点
p->Next = waitNode;
// 插入的结点设为链表尾部结点
waitNode->Next = NULL;

删除结点

删除头指针后的结点

// 创建pp指针，为预删除的结点的下一个结点
struct listNode * pp = newList->Next->Next;
// 删除第一个结点
delete(newList->Next);
// 连接两个结点
newList->Next = pp;

删除链表中间的结点

// 假设删除p结点后的一个结点，则pp为为预删除的结点的下一个结点
struct listNode * pp = p->Next->Next;
// 删除结点
delete(p->Next);
// 连接两个结点
p->Next = pp;

删除链表尾部结点

p = newList;
// 创建p指针，遍历链表使p指针指向链表尾部的前一个结点
while(p->Next->Next!=NULL)
p  = p->Next;
// 删除结点
delete(p->Next);
// 连接两个结点
p->Next = NULL;

输出第n个结点

listNode * outNode(listNode * p, int n)
{
    while(n--)
        p = p->Next;
    return p;
}

cout <<outNode(newList,3)->Data << endl;

链表输出

// 定义一个新的辅助指针用于输出链表
struct listNode * p = newList;
while(p->Next!=NULL)
{
    p = p->Next;
    cout << p->Data;
}
