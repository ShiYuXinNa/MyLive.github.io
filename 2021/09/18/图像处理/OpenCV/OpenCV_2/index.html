Canny边缘检测

  1. 使用高斯滤波器，以平滑图像，滤除噪声。
  2. 计算图像中每个像素点的梯度强度和方向。
  3. 应用非极大值抑制，以消除边缘检测带来的杂散响应。
  4. 应用双阈值检测来确定真实和潜在的边缘。
  5. 通过抑制孤立的弱边缘最终完成边缘检测。

非极大值抑制

  1. 线性插值法： M（q） = w * M（g2） + （1-w）* M（g1）
    w = distance（q,g2) / distance (g1,g2)
  2. 为了简化计算，由于一个像素周围有8个像素，把一个像素的梯度方向离散为8个方向，这样只需要计算前后，不用插值。

双阈值检测

  • 梯度值>maxVal:处理为边界
  • minVal<梯度值<maxVal：连有边界保留，其他舍弃
  • 梯度值<minVal：舍弃

Code

 minVal,maxVal值越小图像，线条越丰富，噪音点越多。值越小，线条越少。

图像轮廓

轮廓检测

 输入的图像应是8位单通道图像，且已转化成二值图像

Mode

  • RETR_EXTERNAL：只检索最外面的轮廓；
  • RETR_LIST：检索所有的轮廓，并将其保存到一条链表当中；
  • RETR_CCOMP：检索所有的轮廓，并将他们组织为两层：顶层是各部分的外界边界，第二层是空洞的边界；
  • RETR_TREE：检索所有的轮廓，并重构嵌套轮廓的整个层次；

 轮廓检索模式：推荐使用最后一种检测模式

Method

  • CHAIN_APPROX_NONE：以Freeman链码的方式输出轮廓，所有其他方法输出多边形（顶点的序列）
  • CHAIN_APPROX_SIMPLE：压缩水平的、垂直的和斜的部分，函数只保留他们的终点部分

模板匹配

 将模板放到原图上进行一个像素一个像素对比，然后将在每个像素点模板对比的度量值生成一个矩阵，根据模板匹配模式选择最相关的像素点。假如原图形是 A×B 大小，而模板是 a×b 大小，则输出结果的矩阵是 （A-a+1）×（B-b+1）（见 参考文档）

Mode

  • TM_CCOEFF：计算相关系数，计算出来的值越大，越相关；
  • TM_SQDIFF：计算平方不同，计算出来的值越小，越相关；
  • TM_CCORR： 计算相关性，计算出来的值越小，越相关；
  • TM_CCORR_NORMED： 计算归一化相关系数，计算出来的值越接近1，越相关；
  • TM_SQDIFF_NORMED：计算归一化平方不同，计算出来的值越接近0，越相关；
  • TM_CCORR_NORMED： 计算归一化相关性，计算出来的值越接近1，越相关；

直方图

直方图定义

 统计一张图片每个像素值有多少个像素点。

  • img：原图像图像格式为uint8或float32。当传入函数时应用中括号[]括来例如[img]
  • channels：同样用中括号,它会告知函数我们这个幅图像的直方图。如果输入图像是灰度图，它的值就是[0]。如果是彩色图像的传入的参数可以是[0][1][2],它们分别对应着B G R
  • mask：掩模图像。统整幅图像的直方图就设他为None。但是如果你想统图像某—分的直方图，你就制作—个掩模图像并使用它。
  • histSize: BIN的数目。也应使用中括号。
  • ranges：像素值范围常为[0,256]

自适应均衡化

  • clipLimit：颜色对比度的阈值
  • titleGridSize：进行像素均衡化的网格大小，即在多少网格下进行直方图的均衡化操作

傅里叶变换

 比起时域，更注重频域分析。

傅里叶变换的作用

  • 高频：变化剧烈的灰度分量，例如边界
  • 低频：变化缓慢的灰度分量，例如一片大海

滤波

  • 低通滤波器：只保留低频，会使得图像模糊
  • 高通滤波器：只保留高频，会使得图像细节增强

  • opencv中主要就是cv2.dft()和cv2.idft() ，输入图像需要先转换成np.float32格式。
  • 得到的结果中频率为0的部分会在左上角，通常要转换到中心位置，可以通过shift变换来实现。
  • cv2.dft()返回的结果是双通道的（实部，虚部），通常还需要转换成图像格式才能展示（0，255）。
