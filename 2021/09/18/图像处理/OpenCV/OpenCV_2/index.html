Canny边缘检测

  1. 使用高斯滤波器，以平滑图像，滤除噪声。
  2. 计算图像中每个像素点的梯度强度和方向。
  3. 应用非极大值抑制，以消除边缘检测带来的杂散响应。
  4. 应用双阈值检测来确定真实和潜在的边缘。
  5. 通过抑制孤立的弱边缘最终完成边缘检测。

非极大值抑制

  1. 线性插值法： M（q） = w * M（g2） + （1-w）* M（g1）
    w = distance（q,g2) / distance (g1,g2)
  2. 为了简化计算，由于一个像素周围有8个像素，把一个像素的梯度方向离散为8个方向，这样只需要计算前后，不用插值。

双阈值检测

  • 梯度值>maxVal:处理为边界
  • minVal<梯度值<maxVal：连有边界保留，其他舍弃
  • 梯度值<minVal：舍弃

Code:

cv2.Canny(img,minVal,maxVal)

 minVal,maxVal值越小图像，线条越丰富，噪音点越多。值越小，线条越少。

#图像轮廓#

轮廓检测

cv2.findContours(img,mode,method)

 输入的图像应是8位单通道图像，且已转化成二值图像

mode:轮廓检索模式

  • RETR_EXTERNAL：只检索最外面的轮廓；
  • RETR_LIST：检索所有的轮廓，并将其保存到一条链表当中；
  • RETR_CCOMP：检索所有的轮廓，并将他们组织为两层：顶层是各部分的外界边界，第二层是空洞的边界；
  • RETR_TREE：检索所有的轮廓，并重构嵌套轮廓的整个层次；

 推荐使用最后一种检测模式

method:轮廓逼近的方法

  • CHAIN_APPROX_NONE：以Freeman链码的方式输出轮廓，所有其他方法输出多边形（顶点的序列）
  • CHAIN_APPROX_SIMPLE：压缩水平的、垂直的和斜的部分，函数只保留他们的终点部分

Code

# 图像读取
img3 = cv2.imread('img.jpg')
# 图像灰度处理
gray = cv2.cvtColor(img3,cv2.COLOR_BGR2GRAY)
# 阈值处理 转换成二值图
res , thresh = cv2.threshold(gray,127,255,cv2.THRESH_BINARY)
# 获取轮廓参数
contours, hierarchy = cv2.findContours(thresh, cv2.RETR_TREE, cv2.CHAIN_APPROX_NONE)
# 图像复制 ， 否则原图会变
draw_img = img3.copy()
# 传入绘制图像，轮廓，轮廓索引，第几个轮廓 默认-1（全部轮廓） 1：一个图形轮廓...，颜色模式 BGR值，线条厚度
res = cv2.drawContours(draw_img,contours,-1,(0, 0, 255),2)

 轮廓绘制图像应是三通道图像，传入获取的contiours列表，线条宽度如果为-1，则为填充模式。

轮廓特征

# 指定第几个轮廓图形
cnt0 = contours[0]
# 计算面积
c1 = cv2.contourArea(cnt0)
# 计算周长 ， True表示闭合的
c3 = cv2.arcLength(cnt0,True)

轮廓近似

# 指定第几个轮廓图形
cnt0 = contours[1]
# 边界值，可以自己设定，一般用公式，系数越小越接近原轮廓
epsilon = 0.1*cv2.arcLength(cnt0,True)
# 得到轮廓近似后的矩阵
approx = cv2.approxPolyDP(cnt0,epsilon,True)
# 图像复制
draw_im = img3.copy()
# 画轮廓
res2 = cv2.drawContours(draw_im,[approx],-1,(159, 204, 45),2)
# 显示轮廓
cv_show("res",res2)

外接矩形

# 获取轮廓0的左上角坐标及宽高
x, y, w, h = cv2.boundingRect(contours[0])
# 画出轮廓外接矩形
imm = cv2.rectangle(img3,(x,y),(x+w,y+h),(0,0,255),2)

外接圆

# 获取轮廓0的圆心坐标和半径 
(x,y), radius = cv2.minEnclosingCircle(contours[0])
# 坐标和圆心数据整型化
center, radius = (int(x),int(y)), int(radius)
# 画出轮廓外接圆
immm = cv2.circle(img3,center, radius, (0, 0, 255),2)

模板匹配

 将模板放到原图上进行一个像素一个像素对比，然后将在每个像素点模板对比的度量值生成一个矩阵，根据模板匹配模式选择最相关的像素点。假如原图形是 A×B 大小，而模板是 a×b 大小，则输出结果的矩阵是 （A-a+1）×（B-b+1）（见 参考文档）

res = cv2.matchTemplate(img,template,mode)

mode：

  • TM_CCOEFF：计算相关系数，计算出来的值越大，越相关；
  • TM_SQDIFF：计算平方不同，计算出来的值越小，越相关；
  • TM_CCORR： 计算相关性，计算出来的值越小，越相关；
  • TM_CCORR_NORMED： 计算归一化相关系数，计算出来的值越接近1，越相关；
  • TM_SQDIFF_NORMED：计算归一化平方不同，计算出来的值越接近0，越相关；
  • TM_CCORR_NORMED： 计算归一化相关性，计算出来的值越接近1，越相关；

Code：

# 读取原图片灰色图
img = cv2.imread('woman.jpg',0)
# 读取模板灰色图
template = cv2.imread('face.jpg',0)
# 获取模板高和宽
templateH, templateW = template.shape[:2]
# 模板匹配输出相关矩阵
res = cv2.matchTemplate(img,template,cv2.TM_SQDIFF_NORMED)
# 获取矩阵最大值，最小值及相关位置
minVal, maxVal, min_loc, max_loc = cv2.minMaxLoc(res)
# 根据匹配模式 获取左上角坐标
top_left = min_loc
# 获取右下角坐标 shape格式是（H，W） 矩阵像素排列（x，y）
bottom_right = (top_left[0]+templateW,top_left[1]+templateH)
# 在原图像灰度图上画矩形,后两个参数表示颜色和线宽
cv2.rectangle(img,top_left,bottom_right,0,1)
# 输出模板匹配度量值矩阵 121表示1，2，1 一行两列 的第一个图像 
# cmap='gray'：用灰度图表示
plt.subplot(121),plt.imshow(res,cmap='gray')
# 隐藏坐标轴
plt.xticks([]),plt.yticks([])
# 输出原图像带矩阵的灰度图
plt.subplot(122),plt.imshow(img,cmap='gray')
plt.xticks([]),plt.yticks([])
# plt.imshow()函数负责对图像进行处理，并显示其格式。
# plt.show()则是将plt.imshow()处理后的函数显示出来。
plt.show()

图像金字塔

高斯金字塔

向下采样方法（缩小）###

  • 将Gi与高斯内核卷积
  • 将所有偶数行和列去除

       |   1   4   6   4   1 |
       |   4  16  24  16   4 |         
1/16 * |   6  24  36  24   6 |
       |   4  16  24  16   4 |
       |   1   4   6   4   1 |

Code：

cv2.pyrDown(img)

向上采样方法（放大）###

                                        |  10   0  30   0 |
       |  10  30  |      ————\          |   0   0   0   0 |
       |  56  96  |      ————/          |  56   0  96   0 |
                                        |   0   0   0   0 |
  1. 将图像在每个方向扩大为原来的两倍，新增的行和列以0填充。
  2. 使用先前同样的内核（x4）与放大后的图像卷积，获得近似值。

Code：

v2.pyrUp(img)

拉普拉斯金字塔

公式：

 Li = Gi - PyrUp(PyrUp(Gi))

Code：

down = cv2.pyrDown(img)
down_up = cv2.pyrUp(down)
L1 = img - down_up
